# CSS Classes in Blazor Accordion Component

The following table provides the list of CSS classes and their descriptions for the Syncfusion Blazor Accordion component:

| CSS Class | Description |
|-----------|-------------|
| `e-accordion` | Applied to the root element of the Accordion component. |
| `e-acrdn-item` | Applied to the wrapper element for an individual Accordion item that contains the header and panel. |
| `e-acrdn-header` | Applied to the header element that toggles the panel when activated. |
| `e-acrdn-header-content` | Applied to the container that holds the header content. |
| `e-acrdn-header-icon` | Applied to the wrapper for header icon elements. |
| `e-toggle-icon` | Applied to the container for the expand or collapse icon inside the header. |
| `e-acrdn-panel` | Applied to the collapsible content panel of an Accordion item. |
| `e-acrdn-content` | Applied to the inner wrapper for content placed inside an Accordion panel (`e-acrdn-panel`). |
| `e-expand-state` | Applied when an Accordion item or its panel is in the expanded state. |
| `e-selected` | Applied to indicate the currently selected Accordion item. |
| `e-nested` | Applied when an Accordion is nested inside another Accordion component. |
| `e-rtl` | Applied when the Accordion component is rendered in right-to-left (RTL) direction. |

## Usage

You can use these CSS classes to customize the appearance of the Accordion component by overriding the default styles in your application's CSS file.
